# 1만 시간의 법칙 ⏰

"연습은 어제의 당신보다 당신을 더 낫게 만든다."

## 📌 프로젝트 소개

**1만 시간의 법칙** 계산기는 어떤 분야의 전문가가 되기 위해 필요한 훈련 기간을 계산해주는 인터랙티브 웹 페이지입니다. 말콤 글래드웰의 저서 "아웃라이어"에서 소개된 1만 시간의 법칙을 기반으로, 사용자가 입력한 하루 훈련 시간을 통해 전문가가 되기까지 필요한 일수를 계산합니다.

## ✨ 주요 기능

- 🎯 전문가가 되고 싶은 분야 입력
- ⏱️ 하루 훈련 시간 입력 및 자동 계산
- 📊 1만 시간 달성에 필요한 일수 표시
- 🎉 응원 모달 팝업
- 📱 반응형 디자인 (모바일 최적화)

## 🛠️ 기술 스택

- **HTML5**
- **CSS3**
  - CSS Variables (커스텀 속성)
  - Flexbox 레이아웃
  - 반응형 미디어 쿼리
- **JavaScript** (modal.js)
- **Google Fonts**
  - Gowun Dodum (고운돋움)
  - Noto Sans KR

## 🎨 디자인 특징

### 컬러 팔레트
- **배경**: `#18181B` (다크 블랙)
- **서피스**: `#27272A` (다크 그레이)
- **포인트**: `#F472B6` (핑크)
- **메인 텍스트**: `#FAFAFA` (화이트)
- **서브 텍스트**: `#A1A1AA` (그레이)

### 반응형 디자인
- 모바일 환경(480px 이하)에서 폰트 크기 자동 조정
- 작은 화면에서는 클릭 아이콘 숨김 처리

## 🚀 사용 방법

1. 전문가가 되고 싶은 분야를 입력합니다 (예: 프로그래밍)
2. 하루에 훈련할 시간을 입력합니다 (예: 5시간)
3. "나는 며칠 동안 훈련을 해야 1만 시간이 될까?" 버튼을 클릭합니다
4. 결과 확인 및 "훈련하러 가기" 또는 "공유하기" 버튼 활용

## 💡 계산 공식
```
필요한 일수 = 10,000시간 ÷ (하루 훈련 시간)
```
